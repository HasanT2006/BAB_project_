## Setup — Before Running

### 1. Install all libraries
```bash
pip install pandas numpy yfinance requests openpyxl statsmodels scipy matplotlib seaborn
pip install gspread oauth2client          # for Google Sheets export (Person 5)
pip install python-telegram-bot requests  # for Telegram alert (Person 4)
```

### 2. Download AQR data (manual step — required)
1. Go to: https://www.aqr.com/Insights/Datasets/Betting-Against-Beta-Equity-Factors-Monthly
2. Download the Excel file
3. Save it to: `data/raw/aqr_bab_data.xlsx`

### 3. (Optional) Set up Telegram alerts — Person 4
In `person4_statistical_analysis.py`:
- Message `@BotFather` on Telegram → create a bot → copy the token
- Get your chat ID from `@userinfobot`
- Fill in `TELEGRAM_TOKEN` and `TELEGRAM_CHAT_ID`
- Set `SEND_TELEGRAM = True`

### 4. (Optional) Set up Google Sheets export — Person 5
In `person5_screener_and_charts.py`:
- Go to Google Cloud Console → enable Sheets API + Drive API
- Create a Service Account → download the JSON key file
- Share your Google Sheet with the service account email
- Set `GSHEETS_KEY_FILE = "google_credentials.json"`
- Set `EXPORT_TO_GSHEETS = True`

## Run Order

```bash
# Option A: Run everything at once
python run_all.py

# Option B: Run individually (in this order)
python code1_backtest/person1_data_loader.py      # needs aqr_bab_data.xlsx
python code2_screener/person1_fetch_current.py    # downloads from Yahoo Finance (~10 min)
python code1_backtest/person3_bab_portfolio.py    # needs outputs from above
python code1_backtest/person4_statistical_analysis.py
python code2_screener/person5_screener_and_charts.py
```

> **Note:** Person 2's script (`person2_beta_calculation.py`) is a shared module — it is imported by Person 3 and Person 5. Run it directly to self-test: `python code1_backtest/person2_beta_calculation.py`
